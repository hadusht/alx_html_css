#ALX 
SmileSchool - Learn to Smile Like a Pro 😊

A modern, responsive website for learning different types of smiles from professional instructors. Built with HTML5, CSS, and Font Awesome icons.

🚀 Features

    Interactive Navigation: Easy-to-use menu for course, pricing, and login.

    Professional Instructors: Showcase of top smile coaches with achievements.

    Tutorial Sections: Featured smile tutorials with ratings and durations.

    Free Membership: Promotional section highlighting benefits.

    FAQ Section: Answers to common questions about the platform.

    Responsive Footer: Social media links and copyright info.

🛠️ Installation & Setup

    bash
